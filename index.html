AirUB (Work in Progress): An iOS and Android App that provides real time air pollution information from Ulaanbaatar, Mongolia, one of the most polluted cities in the world. This project was built using Flutter, Google's cross platform development library, and will be uploaded to the Google Play Store very soon.

Using Machine Learning to Predict All-NBA Selections from the 2019 NBA Draft Class: In this semester long project, I built three machine learning models that estimate the probability of an NBA draftee making an All-NBA team throughout their careers, given their college basketball statistics.

Note (Work in Progress): This is a very simple and minimal notepad application, built using React powered by Material UI. The application uses the web storage API for data persistence across browser sessions.

Bensense: Inspired by NBA All-Star Ben Simmons, this Node.js command line application shows live in-game statistics for an NBA player of the user's choice. If the player manages to complete a triple double, the application notifies the user and exits.

Planets: Although this simple solar system simulator was simply meant to be an introduction project to check Three.js out, I ended up really liking how it looked. You should check it out too!

Sant Academy: In the summer of 2019, I worked at Sant Academy, an education startup in Mongolia, as an instructor and developer. Along with our small team of developers, we built the first prototype website of "NOVA", our flagship platform. The website is currently under development, with plans to ship very soon.

Circles: Inspired by PATATAP, and a project in "The Web Developer Bootcamp" by Colt Steele. This is a simple animated soundboard that makes cool sounds as you press keys on your keyboard. I recommend you check this one out too!
